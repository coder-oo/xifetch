## Xifetch
Neofetch inspired, CLI-tool for view your system information.

### Support:
- [x] Linux
- [ ] MacOS
- [ ] Windows

##

## Installation:
### Clone using Git
```bash
git clone https://github.com/darclinux/xifetch
cd xifetch
python src/main.py
```

### For now you can't install xifetch with any package managers. You should clone this repository.
